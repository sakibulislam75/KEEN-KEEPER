## Overview

**Keen Keeper** is a smart friendship management platform that helps you maintain stronger relationships by tracking communication patterns with your friends. Whether it's calls, texts, or video chats, Keen Keeper logs every interaction and provides insightful analytics to help you understand and nurture your connections at a glance.

---

## ✨ Key Features

- 👥 **Friend Management** — Easily add, view, and manage your friends with detailed profiles including contact information and full interaction history
- 📞 **Communication Tracking** — Log and track interactions across multiple methods (calls, texts, and video calls) with timestamps and detailed records
- 📊 **Analytics Dashboard** — View beautiful, interactive pie charts and statistics that visualize your communication patterns and help you identify which friends you interact with most

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/keen-keeper.git

# 2. Navigate into the project
cd keen-keeper

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open your browser and navigate to `http://localhost:5173` to see Keen Keeper in action.

### Building for Production

```bash
npm run build
```
